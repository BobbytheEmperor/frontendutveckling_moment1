Arbetsproccess

Min arbetsproscess har sett ut på följande vis: Jag har fått lösa en bugg som har lett till ett försvårande av arbetet. Därefter har jag börjat med startsidan för webbplatsen genom att skapa enkel HTML-kod. Simultant har enkel kod för CSS skapats för att förenkla arbetet med webbplatsen. Lorem Ipsum text, alltså placeholdertext har skapats för att underlätta arbetet med grundstrukturen. Innan denna text påbörjades skapades en Javascript-kod för en bildvisning till sidan för bilder. I skrivande stund har publicering ej skett och en del jobb finns kvar på webbsidan. Skapade en develop branch vid den första publiceringen. Använde en pull request för att lägga upp distmappen. Använde sedan merge för att lägga till inneållet i main branch. Först gjordes försök att publicera via Github Pages, vilket inte fungerade och därefter gjordes försök till publicering via Netlify, enligt de instruktioner som getts i teori och läsanvisningarna i en video. När första publiceringen lyckats ändras innehållet på denna webbplats för att möta de krav som gäller för uppgiften, såsom text istället för placeholder texter och ändring utav fotografier till ett modernt filformat.
