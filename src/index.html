Startsida

Uppgiften har gått ut på att börja arbeta med NMP-projekt. Installering utav ett parcelpaket har skett och därefter har uppgiften bestått utav att skapa en enkel webbsida, där html, Javascript samt CSS har använts. Bilder har varit aktuella för uppgiften samt att publicera bilderna i ett modernt format. Publicering av webbplatsen ska ske via exempelvis Github Pages eller Netlify och inte via Miuns servrar. Regelbundna commits till Github har varit en del utav uppgiften.
